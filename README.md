# Wadsworth

## Purpose
A simple discord bot to handle much of the admin work I do for the server as well as a few fun things

## Feature Goals
Here are some fo the features I hope to add:
- **Birthday Tracking**: Have the bot track and wish people a happy birthday
- **Meme Of The Month**: Make it easy to nominate vote and tally votes for favorite memes of every month
- **Reaction Roles**: Assign roles to members based on reactions given to specific messages
- **Quotes**: Make a command to enter a quote into the quotes channel to allow for uniform formatting and legability
